简要题解：

第一题：

首先分离变量，即最小化sigma(x_i * x_j) (i < j) + sigma(y_i * y_j) (i < j).
考虑x的部分，固定除x_i的其他变量，可以发现是关于x_i的一次函数，所以每个x必取到极值处，即-X或X。
sigma(x_i * x_j) (i < j) = (x_1 + x_2 + ... + x_n) ^ 2 - (x_1^2 + x_2^2 + ... + x_n ^ 2)
容易发现后面为定值，即要最小化sigma(x_i)。
经典的背包dp即可


第二题：

一切满足条件的安打率都落在 [r - 10^(-n - 1) * 5, r + 10^(-n - 1) * 5) 之间，而区间的左右端点显然都是有理数。于是我们就要解决这样一个问题：求一个分母最小的有理数 p/q 满足 a/b <= p/q < c/d。
这个做法请见这里：http://wenku.baidu.com/view/40eb1b37a32d7375a41780c9.html
需要注意的是，原文中求得的是最小的有理数满足 a/b < p/q < c/d。所以我们只需要将 a/b 约分之后与 p/q 比较一下即可。


第三题：

我们定义 D(n)为 n 的各位数字的积，那么此题即为求 nD(n)落在 [L, R] 内的 n 的个数。首先，显然有 D(n)<= n，于是 D(n)^2 <= nD(n)<= R <= 10^18，所以 D(n)<= 10^9。预处理所有可能的 D(n)，以及组成这个 D(n)的数字组合（数字组合中不出现 1，这样的话只有不到三十万种可能情况）。然后我们只需要计算 nD(n)<= R 和 nD(n)<= L - 1 的 n 的数目相减即可。对于一个 nD(n)<= R 的问题，我们从 R 的高位向低位运算，把数字组合中的数或者 1 向里面填。到了某一位如果已经小于 R，那么直接 O(1) 计算一下后面的方案数。否则继续向内填充。当然了，n 的位数可以小于 R，所以我们还要对所有比 R 的位数小的 n 进行统计。这样的话对于一个数字组合需要 O(logR) 的时间统计方案数，可以在时限内出解。
